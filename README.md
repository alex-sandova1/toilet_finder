# Delivery Associate assistance
Restroom finder application used to assist Delivery drivers find bathrooms nearby and gas stations that are available to use. 
Allows users to quickly find restrooms during their routes. The restrooms can be 
categorized into public, fast food, restaurant and other types of restrooms

# simple Roadmap
Login → Map → Tap bathroom → Bathroom Details → Navigate → 
After visit: 👍 Used 🚫 Closed 🧼 Clean

# Navigation
users are now able to quickly find their way to the nearest restroom or the 
restroom they have chosen using Google Maps.

# Categorization
The Restrooms are categorized as Public, fast food, gas station, coffee shop, restaurant,
bar or mall restrooms to give the user a large selection and the closest restroom to go to.
